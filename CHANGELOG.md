# 更新日志

## [1.0.0] - 2025-02-11

### 新增功能

#### 演讲稿生成
- 支持中英文演讲稿智能生成
- 自动优化口语表达和结构

#### 实时录音练习
- 录音可视化
- 实时音量显示
- 支持录音回放

#### 口译评估与反思
- 多维度评估体系
- 详细的错误分析
- 专业的改进建议
- 译后反思指导
- 个性化学习建议
- 进步追踪分析

- 基于React + Material-UI的现代化Web界面
- 微信小程序原生开发支持
- 集成SiliconFlow AI API
- Web Audio API音频处理

1. 生成演讲稿

2. 开始练习
